# Démarrage rapide

## Connexion

1. Accédez à l'URL de votre instance CrisisLab
2. Entrez vos identifiants (login + mot de passe)
3. Cliquez sur **Se connecter**

L'authentification utilise un cookie de session sécurisé (`ttx_session`).

## Rejoindre un exercice

Une fois connecté, les exercices auxquels vous êtes affecté apparaissent sur votre tableau de bord.

1. Cliquez sur l'exercice dans la liste
2. Consultez votre **kit d'accueil** pour comprendre votre rôle et le scénario
3. Attendez le signal de démarrage de l'animateur

## Interface joueur

Pendant un exercice, vous avez accès à :

- **Timeline** : historique des injects reçus
- **Canaux simulés** : messagerie, SMS, réseau social, TV...
- **Annuaire de crise** : contacts d'urgence disponibles

## Changer son mot de passe

Rendez-vous dans **Profil** → **Sécurité** → **Changer le mot de passe**.
