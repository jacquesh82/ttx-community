# Injects

Les **injects** sont les stimuli envoyés aux participants pendant un exercice pour simuler des événements de crise.

## Types d'injects

| Type | Canal | Description |
|------|-------|-------------|
| `message` | Messagerie interne | Message texte direct |
| `email` | Webmail simulé | Email dans la boîte simulée |
| `sms` | SMS simulé | Message court |
| `call` | Appel simulé | Simulation d'appel téléphonique |
| `social` | Réseau social | Publication sur le réseau simulé |
| `press` | Presse | Article de presse simulé |
| `tv` | TV | Passage TV simulé |
| `document` | Document | Fichier ou document partagé |

## Réception d'un inject

Quand un animateur envoie un inject :

1. Une notification apparaît dans l'interface joueur
2. Le contenu apparaît dans le canal correspondant (messagerie, webmail, etc.)
3. L'inject est enregistré dans la timeline

## Banque d'injects

Les animateurs peuvent créer des **kits d'injects réutilisables** dans la banque d'injects, organisés par catégorie et thème.
