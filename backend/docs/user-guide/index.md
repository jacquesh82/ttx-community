# Guide utilisateur — Vue d'ensemble

Ce guide s'adresse aux utilisateurs de CrisisLab : **participants**, **observateurs** et **animateurs** d'exercices de crise.

## Rôles utilisateur

| Rôle | Description |
|------|-------------|
| **Participant** | Joue l'exercice, reçoit les injects, répond via les canaux simulés |
| **Observateur** | Suit l'exercice en lecture seule |
| **Animateur** | Pilote l'exercice, envoie les injects, gère la timeline |
| **Administrateur** | Gère la plateforme, les utilisateurs et la configuration |
