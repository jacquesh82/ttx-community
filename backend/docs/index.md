# CrisisLab — Documentation

Bienvenue dans la documentation officielle de **CrisisLab**, la plateforme de simulation de crise.

CrisisLab est une plateforme multi-tenant permettant d'organiser et d'animer des exercices de crise (*Table Top Exercises*). Elle offre un environnement simulé complet avec une timeline d'injects, des canaux de communication simulés, un pilotage de crise et des kits d'accueil personnalisés.
