# Guide administrateur — Vue d'ensemble

Ce guide s'adresse aux **administrateurs** de la plateforme CrisisLab.

## Responsabilités

L'administrateur gère :

- La configuration générale de la plateforme (organisation, logo, options)
- Les comptes utilisateurs (création, rôles, équipes)
- Le serveur d'envoi d'emails (SMTP)
- Les plugins et simulateurs activés
- La sécurité (politiques de mots de passe, sessions)
