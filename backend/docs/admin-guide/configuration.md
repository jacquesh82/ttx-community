# Configuration

## Interface d'administration

Accédez à **Administration** → **Options** dans la barre latérale.

## Organisation

| Paramètre | Description |
|-----------|-------------|
| Nom de l'organisation | Affiché dans l'interface et les documents |
| Logo | Logo de l'organisation (PNG/SVG recommandé) |
| Fuseau horaire | Fuseau utilisé pour les timestamps |

## Options d'exercice

| Paramètre | Description |
|-----------|-------------|
| Durée par défaut | Durée suggérée lors de la création d'exercices |
| Rôles disponibles | Rôles autorisés dans les exercices |
| Plugins activés | Simulateurs et modules optionnels |

## Serveur email (SMTP)

Configurez le serveur SMTP pour l'envoi des kits d'accueil et notifications :

| Paramètre | Description |
|-----------|-------------|
| Hôte SMTP | Adresse du serveur |
| Port | 25, 465 (SSL) ou 587 (TLS) |
| Nom d'utilisateur | Compte d'envoi |
| Mot de passe | Mot de passe du compte |
| Email expéditeur | Adresse affichée dans les emails |

## Sécurité

| Paramètre | Description |
|-----------|-------------|
| Politique de mot de passe | Longueur minimale, complexité |
| Durée de session | Expiration automatique des sessions |
| Tentatives de connexion | Nombre max avant blocage |
