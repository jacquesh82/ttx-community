# Gestion des utilisateurs

## Créer un utilisateur

1. Accédez à **Administration** → **Utilisateurs**
2. Cliquez sur **Nouvel utilisateur**
3. Renseignez : nom, prénom, email, login, mot de passe, rôle
4. Cliquez sur **Créer**

## Rôles

| Rôle | Accès |
|------|-------|
| `admin` | Accès total à la plateforme |
| `animateur` | Créer et piloter des exercices |
| `observateur` | Consulter les exercices en lecture seule |
| `participant` | Jouer les exercices auxquels il est affecté |

## Équipes

Les utilisateurs peuvent être regroupés en **équipes** pour les exercices multi-groupes :

1. Accédez à **Administration** → **Équipes**
2. Créez une équipe et ajoutez-y des membres
3. Lors de la création d'un exercice, affectez les équipes

## Affecter un utilisateur à un exercice

Dans un exercice, section **Participants** :

1. Cliquez sur **Ajouter un participant**
2. Sélectionnez l'utilisateur
3. Choisissez son rôle dans l'exercice (animateur, observateur, joueur)
4. Configurez ses permissions par canal si nécessaire

## Réinitialiser un mot de passe

1. Accédez à la fiche de l'utilisateur
2. Cliquez sur **Réinitialiser le mot de passe**
3. Un nouveau mot de passe temporaire est généré
